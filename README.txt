咪兔写作助手是一款辅导网络写手的小说编写软件，通过便捷的操作、强大的功能，让写手的创作之路更舒心更健康


特色：
    · 便捷的编写界面
      可自动缓存当前输入的内容，彻底解决内容因故障导致丢失问题。
      可智能分析文章内容，滚动到输入位置，无需手动滑动
      
    · 智能设定功能
      预设四种设定类型
      可根据实际需求更改设定类型
      预置常见套路设定
      可按照剧情发展展示各种角色以及出现的设定内容
      
    · 时间线贴合功能
      智能识别文中时间描述，按照章节顺序标记时间线，解决时间线错乱问题
      可手动关闭错误或者无效时间线提示
      
    · 可升级的敏感词库
      自动在线升级敏感词库
      
      
    · 剧情场景卡功能
      想好的剧情可以随时添加进章节、卷的剧情卡中
      在编写章节时可随时查看剧情卡，方便快捷
      
      
    · 书籍备份功能
      可将书籍导出后离线备份，在需要的时候再次导入
      
      
    · 书籍导入导出功能
      可导入备份包
      可导入第三方软件编写的文本
      
      
    · 在线数据同步功能
      支持将所有数据自动备份到服务器
      支持多设备登录，一键同步设置与习惯，更换电脑和设备将不再是问题
      
      
    · 数据统计功能
      每日总量统计、每时效率统计、章节字数统计等多项统计数据
      让您的创作变得明明白白
      
      
    · 健康管理功能
      相比于效率，身体健康更重要，可通过设定智能提醒您进行锻炼、喝水、睡眠等
      
      
    · 快速的升级迭代速度
      平均每周一个版本的速度，让我们的软件始终保持在最新状态
